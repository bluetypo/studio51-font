# Archivos Tipográficos Compilados (`fonts/`)

Coloca en esta carpeta los archivos compilados listos para instalación y uso web de la tipografía **studio51**:
- `studio51-Thin.otf` / `.ttf`
- `studio51-Regular.otf` / `.ttf`
- `studio51-Bold.otf` / `.ttf`
- Formatos para web: `.woff`, `.woff2`

Si compilas la tipografía desde la carpeta `sources/`, guarda los archivos resultantes aquí para publicarlos en GitHub.
