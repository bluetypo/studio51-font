# The studio51 Font Project

**studio51** es una familia tipográfica derivada (*Modified Version*) del proyecto [Montserrat](https://github.com/JulietaUla/Montserrat), creada y adaptada por **Manuel Marquez Alarcón** ([studio51arch.com](https://studio51arch.com)).

Esta tipografía rescata y preserva la esencia urbana y geométrica de su diseño original, evolucionando e incorporando adaptaciones específicas para la identidad, arquitectura y proyectos de **studio51**.

---

## 🔠 Pesos y Estilos Incluidos

La familia **studio51** está compuesta por los siguientes 5 pesos principales, disponibles tanto en versión **Roman (Normal)** como en **Itálicas (Cursivas)**:

| Peso | Roman (Normal) | Itálica (Cursiva) |
| :--- | :--- | :--- |
| **Light** | `studio51-Light` | `studio51-LightItalic` |
| **Regular** | `studio51-Regular` | `studio51-Italic` |
| **Medium** | `studio51-Medium` | `studio51-MediumItalic` |
| **SemiBold** | `studio51-SemiBold` | `studio51-SemiBoldItalic` |
| **Bold** | `studio51-Bold` | `studio51-BoldItalic` |

---

## 🛠️ Archivos Fuente (`sources/`)

En la carpeta `sources/` encontrarás los archivos maestros de diseño en formato `.glyphs`:
- **`studio51.glyphs`**: Contiene la familia multipeso estándar Roman (desde *Thin* hasta *Black*).
- **`studio51-Italic.glyphs`**: Contiene la versión cursiva / itálica en todos sus pesos correspondientes.

Para editar o modificar la tipografía, puedes abrir estos archivos utilizando la aplicación **Glyphs** (macOS) o **FontLab**, o bien compilarla mediante herramientas de código abierto como [`fontmake`](https://github.com/googlefonts/fontmake).

---

## 💻 Archivos Compilados (`fonts/`)

La carpeta `fonts/` alberga las fuentes listas para su instalación en sistemas operativos y servidores web:
- **`fonts/` (Archivos de escritorio OTF):** Contiene los archivos `.otf` (`studio51-Regular.otf`, `studio51-Bold.otf`, etc.) ideales para diseño gráfico, CAD y uso en macOS/Windows.
- **`fonts/woff/` (Archivos Web Font):** Contiene las versiones web (`.woff`) optimizadas para su implementación y carga ultrarrápida en sitios web como [`studio51arch.com`](https://studio51arch.com).
