# Archivos Fuente (`sources/`)

Esta carpeta alberga los archivos de diseño maestro en formato `.glyphs`:
- `studio51.glyphs` (Pesos Normales / Romanos)
- `studio51-Italic.glyphs` (Pesos Cursivos / Itálicas)

Estos archivos son editables con [Glyphs App](https://glyphsapp.com/) o [FontLab](https://www.fontlab.com/), y contienen los trazados, anclas, métricas, kerning e instancias de interpolación de toda la familia tipográfica **studio51**.
